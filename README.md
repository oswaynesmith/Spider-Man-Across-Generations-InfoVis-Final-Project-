## Spider-Man: Across Generations

Spider-Man Across Generations is an interactive data visualization project that explores the legacy of the Spider-Man film franchise across three eras: Tobey Maguire (2002–2007), Andrew Garfield (2012–2014), and Tom Holland (2017–2021). Using D3.js and scroll-driven storytelling, the project uses a timeline, grouped bar ratings charts, a radar chart, and box office comparisons to let users compare each era's reception and performance side by side. 

Developing this project taught us:
- How to identify a story within a dataset.
- How to properly present a data story through combining several visualizations.
- How to work with data in different formats
